# V2.5 Pre-implementation Test Protocol — Gesture Experience Validation

Required by the V2.5 scope before gate review. Implementation may proceed against this
draft; **human test runs may not start until the owner approves or amends it.**

## Hypothesis under test

A first-time visitor can discover and reliably use *open palm → point → pinch to pull a
photograph from the wall* after at most one short visual demonstration, and describes it
as taking/pulling a photograph rather than "clicking with my hand".

## Sample

- Minimum **5 first-time testers**; none may have seen the app before.
- At least 2 with no 3D/gaming background.
- Record age band, handedness, and webcam familiarity per tester.

## Conditions

- **Lighting:** one run in ordinary indoor light (~300 lux or brighter), one run in dim
  evening light per tester. Note backlighting if present.
- **Hardware:** Windows laptop with 720p+ webcam, Chrome or Edge (latest), the same
  build for every tester. Additional devices welcome but recorded separately.
- **Distance/posture:** seated, hand raised no higher than shoulder, ~50–80 cm from camera.

## Measures and targets

| Measure | Target |
|---|---|
| Complete engage → point → pinch → detail without verbal coaching (one ≤20 s demo video allowed) | ≥ 80% of testers within 90 s |
| Accidental activations (unintended selection/engagement) | ≤ 1 per tester per 2-minute browse |
| Tracking-loss recovery (hand removed mid-gesture) | 100% — no stuck selection/zoom state |
| Post-task description | ≥ 60% use take/pull/grab language unprompted |
| Denied-permission and no-camera runs | complete pointer journey retained, 100% |
| Frame rate with gesture active | within the V1 budget on the test laptop |

## Abort criteria

Stop the study and return to design if 3 of the first 4 testers fail the completion
target, or any tester experiences a stuck interaction state that requires a page reload.

## Records

Per tester: condition matrix, timings, activation counts, quotes (anonymised), and any
deviation from protocol. Results land in `docs/evidence/V2.5.md`.
